# ue19-lab-05
# Description
 Script permettant d'appeler un API de blague. On utilise la bibliothèque requests, qui nous permet
 d'envoyer des requetes HTTP à un service en ligne et de récuperer une réponse en format JSON.
 Le code affichera dans le terminal la blague récuperer de l'API.
 Les erreurs potentielles sont vérifiées par le code pour l'empecher de planter.
# How to install and run the project
 copier le code dans un fichier python et faites attetion de bien installer les obligances qui 
 se trouvent dans le fichier requirements.txt. 
 Dans votre terminal tapper la commande "pip install request".
 Ensuite lancer le code avec la commande "python app.py" en prenant garde de bien
 vous trouvez dans le répertoire où se trouve votre fichier python..
